# omni-forge-novel — AI literary writing system from four forges

> FORGE+C 2.0 + OIF Infinity Loop + Nacharium Council — a unified literary engine with 738 techniques, 12 advanced methods, and micro-beat rendering. Works with all major LLMs.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
Omni Forge Novel V1.0 distills Omni Forge V2+V3, OIF V2, and METAFORGE V2 into one coherent literary system. It pairs a structured FORGE+C 2.0 workflow with the OIF Infinity Loop and a council of writing personas (Nacharium) to push prose toward literary quality. Micro-beat rendering keeps each unit tight at 400–600 words. Works with Claude, GPT, Gemini, Qwen, and all major LLMs. Vietnamese + bilingual support.

## Features
- **FORGE+C 2.0** — 7-step writing workflow
- **OIF Infinity Loop** — 6-phase iterative refinement
- **Nacharium Council** — 5 quick roles / 13 full writing personas
- **Micro-Beat Rendering** — 400–600 words/beat
- **Deep toolkit** — 12 advanced techniques, 738 literary techniques, 96 story elements
- **8 prompt templates** + 16 diagnosis patterns
- **14 shortcut commands** + 5 OIF commands
- **Fault tolerance** + self-absorption evolution

## Usage / Quick Start
Choose a starting point: FORGE+C for structured chapter writing, OIF Infinity Loop for iterative refinement, or Nacharium Council for persona-driven prose. All modes render at 400–600 words/beat.
