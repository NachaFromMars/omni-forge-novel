# Checklist Tổng Hợp

> Tất cả checklists: trước viết, sau viết, file naming, tracking.

---

## Checklist Trước Khi Viết (7 mục)

- [ ] **Voice Kit** — Đã paste đoạn mẫu đúng tần số
- [ ] **Skeleton Key** — 1 câu duy nhất chương này tồn tại vì
- [ ] **Emotional Map** — Đường cong cảm xúc với đỉnh + thung lũng
- [ ] **Tempo Script** — Bản nhịp (nhanh/chậm/im lặng)
- [ ] **Constraints** — 3 PHẢI + 3 CẤM
- [ ] **Cường độ** — Vị trí trong tổng thể, thang 1-10
- [ ] **Micro-outline** — 10-15 beats, mỗi beat = 1 câu

---

## Checklist Sau Khi Viết (10 mục)

- [ ] **Adversarial Edit** — 5 tiêu chí + Humanize Pass
- [ ] **Ghost Reader** — 3 câu hỏi, chỗ chán, hình ảnh ám ảnh
- [ ] **Elevate** — 3 câu hay nhất → nâng toàn bộ
- [ ] **Cross-check** — Nhân vật, worldbuilding, timeline vs chương trước
- [ ] **Resonance Audit** — Motif đúng chỗ, đúng nghĩa mới
- [ ] **Skeleton Key test** — Mỗi câu phục vụ? Câu nào không → cắt
- [ ] **Cắt ≥ 15%** — So với draft đầu
- [ ] **Council Nacharium** — Điểm ≥ 7/10
- [ ] **AI Traces** — Không còn dấu hiệu máy viết
- [ ] **Self-Absorption** — Hấp thụ bài học từ report

---

## Checklist Seed Chương Mới (9 mục)

- [ ] Số chương + vị trí trong arc tổng thể
- [ ] Skeleton Key (1 câu)
- [ ] Hạt nhân cảm xúc (1 câu)
- [ ] Intensity Map (cường độ trước/này/sau)
- [ ] Voice Kit reference (tần số nào cho chương này)
- [ ] Resonance motifs cần xuất hiện + nghĩa mới
- [ ] 3 đoạn then chốt từ chương trước (cho cross-check)
- [ ] Foreshadowing cần gieo hoặc gặt
- [ ] Constraints đặc biệt cho chương này

---

## Checklist AI Traces (10 mục)

- [ ] Padding/câu thừa không mang thông tin?
- [ ] Từ "mùi AI"? ("một cách [tính từ]", "không khỏi", "dường như")
- [ ] TELL thay vì SHOW?
- [ ] Giọng trôi so với DNA Sample?
- [ ] Ẩn dụ cliché? (biển, trời, vực)
- [ ] Nhịp đều? (thiếu variation)
- [ ] Thoại phẳng? (không phân biệt nhân vật)
- [ ] Cấu trúc lặp? (mở đầu đoạn giống nhau)
- [ ] Markdown lọt vào prose? (**, #, >)
- [ ] Over-explaining?

**Từ cấm:** tapestry, resonate, nuanced, delve, testament, landscape, multifaceted, intricate

## Checklist Novel Bible (7 mục)

- [ ] Character Sheets (3-5 nhân vật) — wound/want/need/contradiction/3 thoại mẫu
- [ ] World Rules — 5 bất biến + 3 sẽ phá
- [ ] Resonance Map — 5-7 motif + tiến hoá
- [ ] Tone Compass — 1 câu cảm giác
- [ ] Skeleton Key — 1 câu: về gì THỰC SỰ?
- [ ] Micro-Outline 10 chương — tên + POV + beats + cường độ
- [ ] Emotional Map — đường cong toàn tác phẩm

## Checklist Grind (Adversarial Edit + Humanize)

### Adversarial Edit — 5 tiêu chí

1. Câu nào TELL thay vì SHOW?
2. Từ nào lặp trong bán kính 3 câu?
3. Đường cong cảm xúc flat ở giữa?
4. Câu kết ám ảnh hay "kết an toàn"?
5. Cắt đoạn → người đọc mất gì? Nếu không → thừa.

### Humanize Pass — 7 tiêu chí

1. Symbolism phồng?
2. Em dash spam?
3. Từ vựng AI? (tapestry, resonate, nuanced, delve...)
4. Giọng nịnh? (brilliant, stunning)
5. Cấu trúc lặp? (mở đầu đoạn giống nhau)
6. Thiếu personality?
7. Over-explaining?

## Checklist Council Nacharium

### Chế độ nhanh (5 vai)

- [ ] Biên tập trưởng đã đánh giá
- [ ] Nhà phê bình đã đánh giá
- [ ] Độc giả khó tính đã đánh giá
- [ ] Kỹ sư ngôn ngữ đã rà soát AI traces
- [ ] Quản lý liên tục đã cross-check
- [ ] Điểm trung bình ≥ 7/10?

### Chế độ đầy đủ (13 personas)

- [ ] 13 nhận xét (3-5 câu/persona)
- [ ] Điểm tổng /10
- [ ] 3 mạnh + 3 yếu
- [ ] TOP 5 cần sửa
- [ ] Câu hay nhất
- [ ] AI traces check
- [ ] Điểm ≥ 8.5/10?

## Checklist Sau Mỗi Chương (Post-PASS)

- [ ] Cập nhật MASTER-TASK.md (✅ PASS, điểm, số từ)
- [ ] Cập nhật MEMORY-TASK.md (bài học, kỹ thuật mới, điểm yếu)
- [ ] Báo cáo credit: `Ch XX: [time] | [calls] | [credits]`
- [ ] Gửi file FINAL lên Telegram
- [ ] Self-Absorption (ghi memory)
- [ ] Tạo seed chương tiếp

## File Naming Convention

| Loại | Format | Ví dụ |
|------|--------|-------|
| Manuscript FINAL | `"[Tên Tiếng Việt]_Tiểu-Tâm_FINAL.md"` | `"337_Tiểu-Tâm_FINAL.md"` |
| Report Nacharium | `"[Tên] - Đánh Giá Hội Đồng Nacharium.md"` | `"337 - Đánh Giá Hội Đồng Nacharium.md"` |
| Novel Bible | `novel-bible.md` | — |
| Chương raw | `ch[XX]_beat[X].md` hoặc `ch[XX].md` | `ch01_beat3.md` |

**TẤT CẢ FILE:** tiếng Việt có dấu đầy đủ.
